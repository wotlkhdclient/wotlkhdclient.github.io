### **Character Creation Screen - Shadowlands Style**

This patch completely replaces the character creation interface with a redesigned version inspired by World of Warcraft: Shadowlands.

Key changes include:

* Redesigned race and class selection layout with a cleaner sidebar arrangement
* Updated background environments themed around each faction
* All classes feature new armor preview sets inspired by modern WoW expansions

**Related files:** `Data\patch-K.MPQ`, `Data\<lang>\patch-<lang>-K.MPQ`
