This patch replaces the character creation interface with a layout inspired by World of Warcraft: Mists of Pandaria.

Key changes include:

* Race portraits arranged along both sides of the screen with Alliance and Horde separated
* Class selection moved to an icon bar along the bottom of the screen
* Background environments updated to reflect the Pandaria aesthetic - bright, open, and architectural
* All classes feature new armor preview sets inspired by modern WoW expansions
* Overall layout feels more panoramic and open compared to the original 3.3.5a screen

**Related files:** `Data\patch-J.MPQ`, `Data\<lang>\patch-<lang>-J.MPQ`
